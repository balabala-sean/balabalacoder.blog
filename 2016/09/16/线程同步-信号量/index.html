线程同步-信号量

发表于 2016-09-16 | 分类于 并发

信号量(java.util.current.Semaphore)可以控制某个资源在并发的情况下可被同时访问的线程个数，通过 acquire() 获取一个许可，如果没有就等待，通过release() 释放一个许可。

Semaphore(int permits, boolean fair)

  • int permits : 指定线程数
  • fair : 是否采用公平模式，如果以公平方式执行，则线程将会按到达的顺序（FIFO）执行，如果是非公平，则随机唤醒等待的线程。
